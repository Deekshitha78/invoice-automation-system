# Invoice Automation System

This project automates invoice processing using Python and EasyOCR.

## Features
- Extracts vendor name, invoice amount, due date
- Calculates ageing
- Stores data in Google Sheets automatically

## Tools Used
- Python
- EasyOCR
- Pandas
- gspread
- Google Sheets

## Workflow
1. Input invoice images
2. Extract text using EasyOCR
3. Use regex to extract key fields
4. Calculate ageing
5. Store output in Google Sheets
